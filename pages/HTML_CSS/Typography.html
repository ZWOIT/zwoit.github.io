CSS Typography

This is a reference for CSS typography that might be useful. I'm developing this while I learn HTML/CSS. Therefore, this is not exhaustive and only things I use or think might be helpful.

font-family property

The CSS font-family property declares what type of font a rule will use. Multiple fonts can be comma sepparated to designate fallback fonts, in the event the first-due option is unavailble on the computer viewing the webpage. Be sure to use HTML and CSS safe fonts unless serving up a specific font using font-face (described below). Web safe fonts include:

  • Arial (sans-serif)
  • Verdana (sans-serif)
  • Tahoma (sans-serif)
  • Trebuchet MS (sans-serif)
  • Times New Roman (serif)
  • Georgia (serif)
  • Garamond (serif)
  • Courier New (monospace)
  • Brush Script MT (cursive)

Examples

selector {

font-family: Arial;

}


selector {

font-family: Georgia, Arial, serif;

}

font-style property

This is primarily used to set italic and oblique typolography. What's the difference? Italic is technically a different font entirely that is slanted and more caligraphic; oblique is just slanted version of the same font.

Examples

selector {

font-family: Arial;

font-style: italic;

}


selector {

font-family: Arial;

font-style: oblique;

}

font-weight property

Control the thickness or thinness or the text. Default value is normal (400) with the varients thinner, thin, bold (700), and bolder available. Alternatively, numerical values of 100 - 900 can be used.

Examples

selector {

font-family: Arial;

font-weight: thin;

}


selector {

font-family: Arial;

font-weight: 700;

}

line-height, letter-spacing, and word-spacing properties

The line-height property sets what typing apllications refer to as spacing (distance from one line of text to another), while th other two properties are self explanatory. Unitless numbers and any valid CSS units (pixels, percents, ems, rems, etc) are accepted for these properties.

Examples

selector {

font-family: Arial;

line-height: 7;

}


selector {

font-family: Arial;

word-spacing: 4px;

}

@font-face rule

Directly imports external fonts to a stylesheet.This rule should be declared at the top of the stylesheet. The declared font-family value can then be referenced downstream.

Examples

@font-face{

font-family: 'Comic sans';

src: url('../fonts/Comic_sans.ttf') format

('truetype');

}

HTML web font linking

Alternatively from @font-face, the HTML file can reference web fonts if declared in the header. This is useful when using a service like Google Fonts.

Example

<head>

<link href="https://fonts.googleapis.com/css?family=Droid%20Serif" rel="stylesheet">

</head>
